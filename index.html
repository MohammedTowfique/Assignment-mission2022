My Mission in 2022

Mission

In the coming years, MERN Stack is highly likely to be used extensively since more and more applications are now built-in JavaScript. It is the most sought-after language that serves both the front-end and the back-end side, doing away with the need for context switching

Vision

MERN stack is one of the most powerful and in-demand web stacks in 2020. MERN is a free and open-source JavaScript software stack for building dynamic web sites and web applications

Goals

MERN stands for MongoDB, Express, React, Node, after the four key technologies that make up the stack. Before learning MERN, web developers need to get a better understanding of HTML, CSS, and JavaScript. This will help them understand the basic concept of web development and how other stacks can help them.
